# Сцены и ручные инструкции (очень просто)

Эта памятка — «как для чайников». Если нужно просто, быстро и без боли — читай дальше.

## Что это такое

Файл инструкций — это обычный `.txt` файл. В нем ты пишешь команды, а игра их читает по одной строке сверху вниз.

## Как запустить

1. **Создай текстовый файл**, например `instructions.txt`.
2. **Заполни его командами** (примеры ниже).
3. **Укажи файл в `config.yml`:**

```yaml
scene_instructions:
  file: D:\Games\instructions.txt
  encoding: utf-8
  start_paused: false
```

4. **Запусти сервер.**
5. **Когда все готово, NPC начнут выполнять строки по порядку.**

## Самые простые команды

### 1) Сказать фразу

```
сокуциус привет, путник
```

Что значит: NPC с именем/ID, похожим на `сокуциус`, скажет фразу.

### 2) Команда в своем стиле

Коротко — пиши `всс` (это означает «в своём стиле»):

```
сокуциус всс расскажи про Сейда Нин
```

### 3) Прямая цитата

Если нужно, чтобы NPC сказал **буквально** текст (без отсебятины), возьми его в кавычки:

```
сокуциус "Я устал. Я ухожу."
```

### 4) Пауза (стоп-линия)

```
hold
```

Это заморозит дальнейшие строки. Чтобы продолжить — нажми **NumPad 5**.

### 5) Точка интереса (POI)

POI — это «точка на карте». Она пригодится для сцен, где NPC должен куда-то пойти или что-то активировать.

```
poi travel,подойти к двери,-120,450,32
poi activate,открыть дверь,-120,450,32,ex_imp_loaddoor_03
```

- `travel` — просто подойти.
- `activate` — подойти и активировать объект по `ref_id`.

## Полный мини-пример

```
# Комментарии начинаются с #
poi travel,подойти к двери,-120,450,32

сокуциус привет, путник
hold

сокуциус "Я вернулся. Продолжаем."
```

## Частые ошибки (и как не облажаться)

- **NPC не реагирует.**
  - Проверь имя/ID: в строке должно быть что-то, что совпадает с именем NPC.
  - Убедись, что NPC слышит игрока (рядом).
- **Ничего не происходит.**
  - Проверь путь к файлу в `config.yml`.
  - Убедись, что файл сохранён в `utf-8`.
- **После `hold` всё застыло.**
  - Нажми **NumPad 5**.
